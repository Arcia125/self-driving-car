# Self Driving Car

A self driving car simulation that runs in the browser built in typescript.


# Running Locally

```bash
npm run dev
```

# Controls
| Keys      | Description |
| ----------- | ----------- |
| W, Up Arrow      | Forward       |
| A, Left Arrow   | Turn Left        |
| S, Down Arrow   | Reverse        |
| D, Right Arrow   | Turn Right        |
